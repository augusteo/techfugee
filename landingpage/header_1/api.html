Askari API

Restful Web Service Endpoints

Base Url: https://api.askari.xyz


URL: /api/mentors METHOD: GET
Description: Returns the complete list of mentor user entities.
URL: /api/mentees METHOD: GET
Description: Returns the complete list of mentee user entities.
URL: /api/mentors METHOD: POST
Description: Allows submission of a new mentor. Returns the persisted user entity.
URL: /api/mentors?industry=%IndustryString% METHOD: GET
Description: Returns a list of mentor user entities filtered by industry.
URL: /api/mentees?industry=%IndustryString% METHOD: GET
Description: Returns a list of mentee user entities filtered by industry.
URL: /api/user/%UserId% METHOD: GET
Description: Returns the user entity corresponding to the provided UserId.
URL: /api/login METHOD: POST
Description: Uses HTTP Basic Auth to validate username and password parameters for Mentors and Mentees.
URL: api/photo/%UserId% METHOD: POST
Description: Allows for submission of photos for Mentors and Mentees. Returns the updated user entity.
URL: /api/connect/%MentorId%/%MenteeId% METHOD: GET
Description: Initiates a connection request between a Mentor and Mentee. Returns the connection entity.
URL: /api/connect/%UserId%/unconfirmed METHOD: GET
Description: Returns a complete list of connections (confirmed and unconfirmed) for the user corresponding to the UserId.
URL: /api/connect/%UserId%/confirmed METHOD: GET
Description: Returns a list of confirmed connections for the user corresponding to the UserId.
URL: /api/connect/%UserId%/unconfirmed METHOD: GET
Description: Returns a list of unconfirmed connections for the user corresponding to the UserId.
URL: /api/connect/%ConnectionId% METHOD: DELTE
Description: Removes the connection (confirmed or unconfirmed).

URL: /api/connect/confirm/%ConnectionId% METHOD: GET
Description: Marks the Connection corresponding to the ConnectionId as confirmed.
